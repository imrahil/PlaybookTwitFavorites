Twitter Favorites

Twitter Favorites for Blackberry Playbook

Download this project as a .zip file Download this project as a tar.gz file

Simple Adobe Flex app to browse your favorites from Twitter and check URLs in awesome Playbook browser.

Requirements:

  • as3-signals - https://github.com/robertpenner/as3-signals
  • DestroyFramework - https://github.com/imrahil/DestroyFramework
  • TwitterAspirin - https://github.com/imrahil/TwitterAspirin
  • oauth-as3 - http://code.google.com/p/oauth-as3/

Uses VectorCollection from DigitalPrimates - http://www.digitalprimates.net/author/codeslinger/2010/05/18/vectorcollection/

Application icon from http://www.iconfinder.com/icondetails/32131/128/ball_cute_favorites_icon

SkinnableAlert based on Eskimo's one - http://e-skimo.com/
